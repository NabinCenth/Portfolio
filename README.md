A modern personal portfolio website built using React and Vanilla JavaScript with Vite. This project showcases interactive project cards, smooth animations, and a clean, professional design, making it easy to highlight your work and skills online.

Technologies Used

React

Vanilla JavaScript

Vite

CSS (custom styling and animations)

Live Demo

Check out the live website here: bedprasadpokharel.com.np

Installation / Setup (Optional)

If you want to run this project locally:

# Clone the repository
git clone <your-repo-link>

# Navigate to project folder
cd <project-folder>

# Install dependencies
npm install

# Start development server
npm run dev


Then open http://localhost:5173 in your browser.
